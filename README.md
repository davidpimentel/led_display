## Spotify Configuration

## Subway Times

Stop IDs differ based on the direction the train is heading. Here are the currently supported stop IDS:

G Train - Nassau
Northbound: `G28N`
Southbound: `G28S`

L Train - Bedford
Northbound: `L08N`
Southbound: `L08S`

L Train - 1st Ave
Northbound: `L06N`
Southbound: `L06S`
